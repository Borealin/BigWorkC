
1. 目录 
    appcode   ---------  预置的目录，放同学自己的程序代码
    democode   -------- 演示代码
    demoprj-devc ------ 演示程序的dev-c工程文件
    demoprj-vs2010---- 演示程序的visual studio 2010 工程文件
    demoprj-vs2017---- 演示程序的visual studio 2017 工程文件
    documents --------- 文档：
                                     1. 工程文件介绍，
                                     2. ligraphics介绍，
                                     3. simpleGUI介绍
                                     4. 图形编程习题
                                     5. visual studio 入门使用
    libgraphics --------- 图形库libgraphics源程序和头文件
    simpleGUI ---------  基于libgraphics的图形控件的源代码和头文件
    readme.txt---------- 本文件
   
2. 建立和设置visual studio和dev-c的程序工程都不容易，需要经验。
    最简单方法是：

      2.1 复制演示程序工程。注意需要复制完整的目录
      2.2 删除工程中不要的文件 
      2.3 添加自己的程序文件 

    详细内容，参考documents目录中工程文件的教程
